# Nostalgic Trees

Resource trees for Minecraft. Each tree grows into a harvestable source of a specific item — from dirt and stone up through diamond and netherite. Progress through tiers by crafting, mutating, and processing.

This wiki covers configuration and packmaker integration. For general gameplay information, see the [CurseForge page](https://www.curseforge.com/minecraft/mc-mods/nostalgictrees).

## Who this wiki is for

**Players.** Read [For Players](/for-players) for a short gameplay overview.

**Packmakers.** Read [For Packmakers](/for-packmakers) for the integration points — what you can configure, what KubeJS scripts can modify, and what's locked down.

**Modders integrating on the Java side.** This wiki doesn't cover internal APIs. The [source repository](https://github.com/Nostalgic-am/nostalgictrees) is the canonical reference.
